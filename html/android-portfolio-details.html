Favorite restaurants near Seminole State College

Project information

  • Category: Android Mobile Application
  • Technology Used:
  • Project date: Spring,2021
  • Project URL: www.example.com

This application was created for students near Seminole state to be able share their favorite restaurants that were located on campus and be able to select these locations from a map on campus. What does it do?Issues that came up during implementation, there are no restaurants on the campus that could be put into a list, there is only a cafeteria. How it can be improved The other issue was selecting locations on a map and then populating those restaurants into a database, this also was incredibly challenging, we were able at one point to get the user to be able to select locations on the map but not save to a database and not populate the way we wanted to. After several modifications the application displays the top 10 restaurants located near Seminole state College that have the highest rating, the user can then select, their favorite locations and then view a list of their favorites on another page. We also incorporated a map so that the user could view the top 10 restaurants on a map and see how close they are to Seminole state College. Our application utilized database functionality and location via google maps.
